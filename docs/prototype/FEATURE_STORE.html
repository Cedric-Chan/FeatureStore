Online Feature Store 交互演示

交互原型已迁移为 Vite + React 工程（唯一真源）。本页仅作入口说明。

方式 A — 本地开发（热更新）

  1. 在终端进入：docs/references/feature-widetable-figma-export
  2. 执行：npm install 然后 npm run dev
  3. 浏览器打开终端里提示的本地地址（一般为 http://127.0.0.1:5173）。

方式 B — 构建后离线打开 / 打包分享

  1. 在同一目录执行：npm run build（或 npm run build:demo）。
  2. 生成目录 dist/。可将整个 dist 打成 zip 发给他人。
  3. 解压后直接双击 dist/index.html 用浏览器打开（路由使用 hash，无需服务器）。若遇安全策略限制，可在 dist 下执行：python3 -m http.server 8765 后访问 http://127.0.0.1:8765。

直接打开已构建的演示（相对本文件）

打开 dist/index.html

若链接无效，请先完成上述 npm run build 以生成 dist。
