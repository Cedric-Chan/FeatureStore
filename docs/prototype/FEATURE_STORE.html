Online Feature Store 交互演示

交互原型为 Vite + React（仓库内唯一真源）。

推荐：公开网页分享（类似 Figma Published，无需本地 npm）

把本仓库放到 GitHub 后，流水线会自动把最新界面发布成固定网址，发链接给别人即可。

演示地址格式一般为：
https://你的用户名.github.io/你的仓库名/

一步步说明（首次只需设置一次 Pages）

其他方式（可选）

  • 本地开发：进入 docs/references/feature-widetable-figma-export，执行 npm install 与 npm run dev。
  • 离线 zip：同目录执行 npm run build，使用生成的 dist 打包分享。
  • 本文件夹预览：同目录执行 npm run sync-prototype 后，可打开 feature-store-demo/index.html（需本机先执行过同步）。

归档的旧版单文件原型（只读）
